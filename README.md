# Ignite Marketplace

Ignite is a premium e-commerce marketplace built for professional business gadgets, apparel, and executive equipment. This platform is designed as a high-end demonstration of a modern marketplace experience with a focus on dark aesthetics, premium typography, and modular frontend architecture.

## Key Features

- **Premium Marketplace Interface**: A sleek, dark-themed UI built with professional standards using Bootstrap 5 and custom CSS (Glassmorphism).
- **Modular JavaScript Architecture**: All logic is built using ES6 modules (app.js, products.js, cart.js) for clean and maintainable code.
- **Dynamic Marketplace Filtering**: Advanced category, search, and sorting filters implemented in the shop and library sections.
- **Interactive Shopping Cart**: Live cart sidebar with real-time total calculations and persistence.
- **Seller Framework (Demo)**: A mock seller dashboard that allows users to trial store creation and product listing functionality.
- **Secure Payment Integration (Reference)**: Built-in hooks for Paystack and Flutterwave checkout flows.
- **Responsive Design**: Fully optimized for mobile, tablet, and high-resolution desktop displays.

## Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Variables), Modular JavaScript (ES6)
- **Framework & Icons**: Bootstrap v5.3.3, Bootstrap Icons v1.11.3
- **Animations**: CSS3 Reveal Animations, Intersection Observer API
- **Form Handling**: Integrated with Formspree for secure message handling

## Project Structure

- **index.html**: The master landing page and feature showcase.
- **shop.html**: The main product browsing and filterable library.
- **deals.html**: Promotion page for current marketplace offers.
- **signin.html**: Professional authentication interface for members and sellers.
- **inbox.html**: Dedicated contact and support page.
- **src/css/**: Custom design tokens, page-specific styles, and animation modules.
- **src/js/**: Modular application logic and page initialization modules.

## Important Note

This repository serves as a demonstration and portfolio project.
